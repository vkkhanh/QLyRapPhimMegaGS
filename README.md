# 🎬 HỆ THỐNG QUẢN LÝ BÁN VÉ RẠP CHIẾU PHIM MEGA GS

## 📌 Giới thiệu

Hệ thống Quản lý Bán vé Rạp chiếu phim Mega GS được xây dựng nhằm hỗ trợ quản lý các hoạt động vận hành tại rạp chiếu phim, bao gồm quản lý phim, suất chiếu, khách hàng, bán vé và báo cáo doanh thu.


## 🛠 Công nghệ sử dụng

- **Ngôn ngữ lập trình:** C#
- **Giao diện:** Windows Forms
- **Cơ sở dữ liệu:** SQL Server
- **IDE:** Visual Studio

---

## 📂 Chức năng chính

### 🔹 Quản lý phim
- Thêm, sửa, xóa thông tin phim

### 🔹 Quản lý thể loại
- Thêm, sửa, xóa thể loại phim
- Phân loại phim theo thể loại

### 🔹 Quản lý suất chiếu
- Thiết lập lịch chiếu theo phim và phòng

### 🔹 Quản lý bắp – nước
- Quản lý danh sách sản phẩm bắp rang, nước ngọt và combo
- Cập nhật giá bán và số lượng tồn

### 🔹 Quản lý khách hàng
- Lưu trữ và cập nhật thông tin khách hàng

### 🔹 Quản lý nhân viên
- Quản lý thông tin nhân viên
- Thêm, sửa, xóa và tra cứu nhân viên

### 🔹 Báo cáo – thống kê
- Thống kê doanh thu theo thời gian
- Thống kê vé bán và sản phẩm bắp – nước
- Hỗ trợ in danh sách báo cáo hoặc xuất file Excel

### 🔹 Sao lưu và phục hồi dữ liệu
- Sao lưu toàn bộ cơ sở dữ liệu
- Khôi phục dữ liệu khi cần thiết
